# Crowd - Social media prototype.
Free speach social media prototype in development, based in RestFul API.
Most of the project is done.

## What can I do in Crowd?
- Login & Register with JWT.
- Upload "crowds".
- Follow users.

### Technologies used:
- ReactJS
- NodeJS
- MariaDB pulled with Docker.
